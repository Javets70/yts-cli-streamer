# YTSP
It is a CLI movie streaming client which works on yts.mx API written in python.

## Requirements
- requests
- rich
- subprocess
- webtorrent-cli 

## Using the CLI 
```bash
python ./main.py --help
```
This will print the help menu with all the details about **ytsp**.

```bash
python ./main.py "batman"
```
